git init. Инициализация локального репозитория

git init - инициализация локального Git репозитория. Эта команда создаёт в текущей директории новую поддиректорию с именем .git, содержащую все необходимые файлы репозитория. Если вы закините в эту инициализированную папку какой-нибудь файл или модифицируйте файл,то Git будет отслеживать все эти изменения.
