Установка

Заходим на сайт по ссылке https://git-scm.com/downloads и там выбираем на какую операцинную систему вы будете скачивать установщик Git. Далее устанавливаем. На Windows при установке надо поставить галочку на Git Bash Here.
Чтобы узнать установился ли у вас Git и узнать версию нужно в консоли ввести git --version
