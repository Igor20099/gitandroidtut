git commit

Команда которая позволяет делать снимок/слепок проекта. Для начало мы проверяем статус проекта с помошью git status и смотрим если ли untracked и modifed файлы. Далее добавляем их в статус подготовленные с помощью команды git add ..После этого файлы готовы к коммиту
git commit -m "Комментарий" - стандартный вызов команды git commit. Пишем комментарий в ковычках какие изменения были в нашем проекте.
git commit -a -m "Комментарий" - команда которая сразу добавляет в статус подготовленного файла и делает коммит. Работает с modifed(изменным файлом)
git commit --amend -m "Комментарий" - команда которая не добавляет новый коммит, а изменяет последний.
