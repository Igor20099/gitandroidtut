git add - команда которая добавляет untracked(неотслежеваемый файл) и modifed(измененный файл) в статус подготовленного файла.
git add название_файла - добавляет файл в статус подготовленного для коммита. Так же можно добавить еще файлы через пробел "git add название_файла название_файла_2"
git add . - команда git add c точкой добавляет все файлы в статус подготовленных файлов для коммита.
git add *.html - добавляет все файлы с в текущей папке с расширением ".html".
git add Dir/*.html - добавляет все файлы в папке "Dir" c расширением ".html"
git add Dir/ - добавляет все файлы в папке "Dir".
git add "*.html" - добавляет все файлы в проекте с расширением ".html"
