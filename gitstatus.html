git status. Untracked и Modifed файлы

git status - команда которая показывает есть ли файлы как untracked(неотслежеваемый файл) и modified(измененный файл). Если вы только добавили в инициализированную гитом папку новый файл , то этот файл будет как untracked и будет выделен красным цветом. Для того что бы добавить его в статус отслежеваемого нужно выполнить команду git add название_файла. После этого вызвав команду git status вы увидете что файл стал зеленым и готов для коммита.
Так же если вы сделаете изменения в файле при вызове команды git status он будет красным помечен как modified: назание файла.Нужно так же вызвать команду git add название_файла и вызвав команду git status вы снова увидите что надпись стала зеленая и файл готов для коммита.
